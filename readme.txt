Le Widget météo de MétéoMedia pour Wordpress 

== Description ==

Le Widget météo de MétéoMedia pour Wordpress 

Il est plus facile que jamais d'informer vos visiteurs des plus récentes conditions météorologiques. 

Il suffit de téléverser le widget de MétéoMédia sur votre site en Wordpress et l'installer automatiquement pour offrir du contenu météo à jour aux internautes qui visitent votre site.

Disponible en version pâle ou foncée, le widget météo de  MétéoMédia s'intègre parfaitement au design de votre site.



== Installation ==


1. Téléchargez le fichier mm-weather.zip à partir du site suivant: http://www.meteomedia.com/weatherplugin/wordpress/mm-weather.zip
2. Importez et activez le plugin à partir du menu Plugins de WordPress
3. Trouvez le Code d'emplacement pour la ville dont vous désirez les prévisions météo à: http://www.meteomedia.com/weather_centre/wcwxbutton .
