=== Weather Widget MeteoMedia ===
Collaborateurs: MétéoMédia
Tags: météo, température, prévisions, prévisions météo de la ville


== Description ==

Affichez la météo de votre ville sur votre barre latérale. Choisissez parmi différentes mises en page et dessins

* Définissez vos préférences de widgets au http://www.meteomedia.com/widget-meteo
* Choisissez parmi deux couleurs de police différentes
* Choisissez entre métrique ou impérial
* Choisissez entre un ou plusieurs endroits (jusqu'à six)

== Installation ==

1. Téléchargez le fichier Zip et accédez au contenu
2. Télécharger le dossier `mm-meteo` dans le répertoire  "/ wp-content / plugins / " de votre dossier  plugin WP.
3. Allez Plugins> Plugins et activez le plugin
4. Allez à Apparence> Widgets, et faites glisser le widget vers la barre latérale
5. Entrez le ID widget ainsi que le premier code de localisation que vous avez obtenu à partir du http://www.meteomedia.com/widget-meteo dans le menu des paramètres du widget
